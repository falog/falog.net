# falog.net

言語学習（主に英語）、ソフトウェア開発、回路設計、ものづくりの試行錯誤を記録する個人ブログ。

## Overview

このリポジトリは、Astroを使って構築した個人ブログです。
最小構成から積み上げるスタイルで、以下の内容を発信しています。

- 自作CPU（最小構成 / 命令設計）
- RISC-Vに向けた学習ログ
- Web開発（Astro / TypeScript）
- 技術メモ・検証記事
- 英語学習（技術英語・発音など）

## Features

- Astroによる高速な静的サイト
- Markdown / MDXによる記事管理
- SEO対応（OGP / sitemap / RSS）
- シンプルな構成で拡張しやすい設計

## Tech Stack

- Astro
- TypeScript
- pnpm

## Development

```bash
pnpm install
pnpm dev
```

## Build
```bash
pnpm build
pnpm preview
```
